> **"Same NJFL. Zero weapon changes. Perfect for servers that already have their own combat balance."**

Same design as [**Not Just For Looks**](https://github.com/gitpush-mod/se-not-just-for-looks) — buffed DLC reskins, tiered variants, Superconductor tax — but with **all weapon adjustments stripped out**. Made for servers and playgroups that curate their own combat balance and don't want any mod touching weapons or turrets.

</p>

## ✨ What's different vs. NJFL

| | NJFL | NJFL — Without Weapons |
|---|---|---|
| Buffed thrusters, reactors, hydro engines | ✅ | ✅ |
| Buffed refineries, assemblers, cargo | ✅ | ✅ |
| Buffed drills / welders / grinders / ore detectors | ✅ | ✅ |
| Weapon durability + cargo buffs | ✅ | — |
| Turret range buff | ✅ | — |
| Warfare Gatling / Rocket Launcher buffs | ✅ | — |
| Gatling / Rocket Turret Type II | ✅ | — |

## 🚀 Install

1. [Subscribe on the Workshop page](https://steamcommunity.com/sharedfiles/filedetails/?id=3686420788)
2. **Do NOT** also install the regular [NJFL](https://steamcommunity.com/sharedfiles/filedetails/?id=3572467804) — pick one or the other, they'd conflict
3. Enable the mod and build any DLC block — upgraded/advanced variants appear on the variant arrow cycle (weapons untouched)

## 🎯 Compatibility

- ✅ **Space Engineers 1** — actively maintained
- ✅ **Save-safe**
- ✅ **Multiplayer + dedicated servers** — the reason this variant exists
- ⚠️ **Do not stack with regular NJFL** — pick one
